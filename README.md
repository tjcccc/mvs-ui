# MVSUI

A lightweight CSS Framework.

## Installation

```bash
npm install --save mvsui
# or
yarn add mvsui
```

## Usage

### CSS reset only

HTML:

```html
<link rel="stylesheet" href="{node_modules path}/dist/mvsui.reset.min.css" />
```

JS:

```js
import 'mvsui/dist/mvsui.reset.min.css';
```
